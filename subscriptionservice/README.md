# Subscription Management System

A full stack subscription management application built using React and Spring Boot.

## Tech Stack
- React
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Axios

## Features
- Create, list, and delete subscriptions
- Backend validation and global error handling
- Structured logging with SLF4J
- Layered architecture (Controller, Service, Repository)
- Frontend-backend integration with Axios

## How to Run

### Backend
```bash
./mvnw spring-boot:run
